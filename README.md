Hull Breach

Description: Hull Breach is an action-strategy space-combat game where the player commands a heavily armed capital ship navigating hostile deep-space territories. Players Utilize their ship turrets and ammo type to damage or penetrate enemy ships along with navigating a tactical map. 
Solo group, Anthony Nelsen
